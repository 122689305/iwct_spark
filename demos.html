The core source code of PageRank：
var PR = Array.fill(n)( 1.0 )
val oldPR = Array.fill(n)( 1.0 )
for( iter <- 0 until numIter ) { swap(oldPR, PR) for( i <- 0 until n )
{ PR[i] = alpha + (1 - alpha) * inNbrs[i].map(j => oldPR[j] / outDeg[j]).sum } }

1. The second parameter of “PageRank()” can control accuracy and running time.
2. The default of resetProb is 0.15.
